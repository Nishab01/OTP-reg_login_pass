Import Databse:
	1. Create a dtabase named 'user_db'.
	2. Click the database, then click the 'Import' button in the top of the scrren.
	3. Choose the file names 'user_db.sql.gz'. The file will be found inside the folder named 'db'.
	4. Scroll down and click 'import'. 
[Note: Do not change any settings, that have not been mention above.]


Source Code Import:
	1. If you have already installed Xampp in your system. then proceed to the next step. Otherwise, install Xampp, then proceed.
	2. Find the 'htdocs' folder inside the Xampp directory.
	3. Find the 'source_code' folder. Inside the folder, copy the batch file named 'OTP-reg_login_pass.zip'. Copy the batch file, paste it inside the 'htdocs' folder.
	4. Unzip the batch file there ('htdocs' folder).
	5. Run and start the Xampp servers (Apache & MySQL).
	6. Go to this url(localhost/OTP-reg_login_pass) with your browser.
